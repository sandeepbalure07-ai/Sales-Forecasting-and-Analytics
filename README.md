# 📈 Sales Forecasting and Analytics

A Python-based sales analytics project that analyzes historical retail sales data, visualizes business trends, and predicts future sales using Linear Regression. The project also includes an interactive Power BI dashboard for business insights.

---

##  Features

* Sales analysis by Region
* Sales analysis by Category
* Monthly Sales Trend Analysis
* Moving Average Trend Visualization
* Future Sales Forecasting using Linear Regression
* Interactive Power BI Dashboard

---

##  Tech Stack

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Power BI
* NumPy

## 📊 Project Workflow

1. Load and preprocess the Superstore sales dataset.
2. Analyze sales by region.
3. Analyze sales by product category.
4. Visualize monthly sales trends.
5. Calculate Moving Average for trend analysis.
6. Train a Linear Regression model.
7. Forecast future sales.
8. Present insights using Power BI Dashboard.
